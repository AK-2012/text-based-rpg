# Text Based RPG
A Python command-line fantasy RPG with a modular room system and world exploration.

## Overview

You begin as a prisoner locked away in a dungeon beneath a castle ruled by an evil king. As you explore, you'll encounter different rooms, quests, and choices that affect your path. 

Players progress by entering command-line inputs and making decisions.

Examples include:

- Exploring new locations
- Collecting items
- Fighting enemies

## Features

- Modular room-based architecture
- Inventory and item system
- Combat mechanics
- Branching story paths
- Multiple endings
- Persistent game state using flags
- Exploration-focused gameplay

## Installation

Clone the repository:

```
git clone https://github.com/AK-2012/text-based-rpg
```

```
cd rpg
```


### Requirements

- Python 3
- colorama

### Install dependencies: 

```
pip install colorama
```

### Running the RPG

```
python main.py
```
